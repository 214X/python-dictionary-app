# Python Dictionary App

A simple command-line English dictionary manager built with Python. It lets you create, load, and save dictionary files, then add, remove, search, and list words with their part of speech, plural form, and definitions.

## Features

- Create / load / save / remove dictionary files (`.dict`)
- Add words with part of speech (adjective, adverb, noun, verb), plural, and definitions
- Auto-generate plural forms when not provided
- Search and list words, with automatic cross-referencing (underlining) of dictionary words used in other definitions
- Simple menu-driven CLI

## Usage

```bash
python main.py
```

## Story

This project was developed as a practical example to teach Python fundamentals to a new intern at our office. To provide clear, hands-on explanations of core programming concepts, I implemented the entire project myself while following best practices and emphasizing readability, modularity, and maintainability. The project demonstrates fundamental Python concepts such as functions, file handling, data structures, error handling, and basic application design through a simple dictionary management application.
